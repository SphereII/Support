# Support
7D2D Mod Launcher Support

Have trouble with the 7D2D Mod Launcher? Give these a try!


Problem #1:

When you go to update the mod launcher, it starts to throw an error:

ERROR SUMMARY
	Below is a summary of the errors, details of these errors are listed later in the log.
	* Activation of C:\Users\adria\Downloads\7D2DLauncher.application resulted in exception. Following failure messages were detected:
		+ Value does not fall within the expected range.


Solution:

Go into your Add and Remove Programs, and uninstall the existing Mod Launcher, then try again.


Problem #2:

There's no mods listed!

Solution:

If you are running an Anti-virus program, you may have to add the Mod Launcher, and your mod folder, to the exclusion list/

Norton Anti-Virus: http://smallbusiness.chron.com/set-exclusions-norton-antivirus-61906.html
McAfee Anti-Virus: https://itstillworks.com/add-exceptions-mcafee-29665.html


Problem #3:

It's taking a long time to download a mod.

Solution:

There are a few things you can do, if you feel that things are taking a long time, or it appears to be hung.

 - Press Cancel Sync, and try again
 - Press cancel Sync, reboot your PC, and try again
 - If available, check the "Check here to use a mirror", and try again
 - Check the Direct Download, and try again. Disable the Refresh Mods Automatically after it's done.
